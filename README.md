# Drinkit: Анализ обращений

Интерактивный дашборд для анализа обращений Drinkit за период 27 мая - 27 июня 2026.

## 📊 Возможности

- **Категоризация обращений** по типам проблем (Касса, Печать, Меню, Акции и т.д.)
- **График популярности** категорий
- **Иерархическая структура**: Категория → Компонент → Тема
- **Прямые ссылки** на задачи в Pyrus
- **Поиск** по всем полям
- **Сортировка** от популярного к редкому

## 🚀 Просмотр
